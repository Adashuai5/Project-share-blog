# 关于 Vue-Blog

- 该项目是我初学 Vue 时，根据教程自主开发的一个前端项目，其可作为博客分享网站。
 
- 项目使用 Vue.js 技术栈：**vue-cli/vue2/axios/vue-router/vuex/es6/npm**

- 包含首页、用户文章列表、登录、注册、个人管理、编辑、发布等功能。
